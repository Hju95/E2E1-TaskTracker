# E2E1-TaskTracker
공유 일정,목표 관리 서비스
---
## 기술 스택
- Front-end:
    - Bootstrap
    - ThymeLeaf
- Back-end:
    - Java 17
    - Spring Framework
        - Spring Boot 3.0.12
        - Spring Data JPA
        - Spring Security
            - Kakao 로그인
            - OAuth2
    - MySQL 8
    - AWS (Amazon Web Services)
    - Docker
    - Redash
